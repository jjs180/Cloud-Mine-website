DCI HOMES

This project involved the design and development of a new mobile responsive website. We created a project hub to feature real estate development projects and designed a project page template customizable with project-specific branding. CloudMine Creative extended some existing project branding on the new website and developed the brand for Vantage Point, a luxury town home development in Echo Park. The new project template allows for the brand consistent development of new project websites faster and at a reduced cost, but to still exude a custom designed look and feel.
